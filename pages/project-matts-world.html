Role: Developer & Designer
Technologies: HTML, CSS, JavaScript
View Live Site

The Challenge

The goal was to create a simple, engaging, and fun website for a young child to use for learning. The design needed to be colorful and interactive, with a focus on ease of use for a non-technical, young audience.

My Solution

I built a playful, single-page website with a cartoonish island theme. Using simple shapes, bright colors, and animations, I created an environment that encourages exploration. The interactions are straightforward click-based events, perfect for a child. The project was built with vanilla HTML, CSS, and JavaScript, keeping it lightweight, fast, and easy to maintain.
