Almanac Utility Tool

Role: Full-Stack Developer
Technologies: HTML, CSS, JavaScript, API Integration, Gemini API
View Live Site

The Challenge

The goal was to create a compelling, dynamic portal for information related to any given day. Traditional almanacs are static, but this tool needed to provide an interactive, personalized "daily digest" pulling from multiple data sources, such as historical events, celestial data, and trivia.

My Solution

I developed a modern front-end using clean HTML and CSS, driven by JavaScript to handle user interaction (date selection). The core functionality relies on integrating with various public APIs (History, NASA, etc.) to fetch raw data. I utilized the Gemini API to analyze and synthesize this raw information into coherent, easy-to-read, narrative summaries, turning a collection of facts into a rich, daily story.
