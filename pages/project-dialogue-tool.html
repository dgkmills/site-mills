Dialogue Generation Tool

Role: Full-Stack Developer
Technologies: HTML, CSS, JavaScript, Gemini API
View Live Site

The Challenge

Language learners and content creators often need to generate and listen to natural-sounding dialogues. The challenge was to build a tool that could not only create scripts using AI but also convert them into multi-speaker audio with customizable voices, all within a simple, easy-to-use interface.

My Solution

I developed a web-based tool that integrates with the Gemini API for both text and audio generation. The user interface allows users to select topics to auto-generate scripts or write their own. They can then assign different voices to each speaker and generate the audio. The tool is built to be intuitive, guiding the user through the process step-by-step. It's a powerful demonstration of how AI can be leveraged to create practical and creative tools.
