Student Portal (My Teacher Dan)

Role: Full-Stack Developer
Technologies: HTML, CSS, JavaScript, Firebase Authentication
View Live Site

The Challenge

The 'My Teacher Dan' website needed a secure, members-only area where enrolled students could log in to access their lesson materials, watch exclusive videos, and view their personalized dashboard. This required a robust authentication system and a dedicated front-end, separate from the public-facing site.

My Solution

I built a single-page application and secured it using Firebase Authentication. Students are given login credentials, and upon successful login, they are granted access to the portal. The portal itself is a dashboard that dynamically displays lessons, videos, and other resources. This provides a professional and secure learning environment for students, all while running on Firebase's free-tier "Spark Plan."
