Aree's Site (Student Site)

Role: Instructor & Developer
Technologies: HTML, CSS, JavaScript
View Live Site

The Challenge

This project was part of the 'Web Kid$' class, designed to teach students the fundamentals of web development. The goal was for the student, Aree, to create a personal portfolio website to showcase her interests and projects.

My Solution

I guided the student through the process of designing and building a simple, static website using HTML and CSS. The project focused on fundamental concepts, clean code, and personal expression. The result is a charming personal site that serves as a testament to the student's new skills and creativity.
